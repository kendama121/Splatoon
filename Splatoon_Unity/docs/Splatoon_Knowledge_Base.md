## 1. Splatoonとは(概要・歴史)

### 1.1 シリーズ基本情報

| 項目 | 内容 |
|------|------|
| ジャンル | チームベース三人称シューター(TPS) + アクション |
| 開発・発売 | 任天堂 |
| 創始者 | 野上恒(Hisashi Nogami)、佐藤慎太郎(Shintaro Sato) |
| 特徴 | キル数ではなく**塗り面積**で勝敗を決める独自ルール |

### 1.2 シリーズ作品

| 作品 | プラットフォーム | 発売日 | 主な追加要素 |
|------|-----------------|--------|-------------|
| Splatoon | Wii U | 2015/05/28 | シリーズ初作。ナワバリバトル、ヒーローモード「タコツボバレー」 |
| Splatoon 2 | Switch | 2017/07/21 | サーモンラン追加。テンタクルズMC |
| Splatoon 2: Octo Expansion(DLC) | Switch | 2018/06/13 | 8号主人公、オクトリングプレイアブル化 |
| Splatoon 3 | Switch | 2022/09/09 | バンカラ街、新カテゴリ(ストリンガー/ワイパー)追加 |
| Splatoon 3: Side Order(DLC) | Switch | 2024/02/22 | ローグライク、Side Order |

### 1.3 ゲームコンセプト

- **4vs4のオンライン対戦** が中心
- インクを撃つ → 地面が塗られる → **塗り面積** で勝敗
- イカ形態(ZL)で自軍インク内を**高速移動**、人型形態(ZR)で**射撃**
- 14歳以上のインクリング/オクトリングが「ナワバリバトル」をスポーツとして楽しむ世界

## 2. 世界観・歴史

### 2.1 軟体世紀(なんたいせいき)

舞台は **約12,000年後の地球**。海面上昇により人類と哺乳類は絶滅し、進化した海洋軟体生物が地上文明を築いた **ポストアポカリプス世界**。

- ゲーム内紀年法 = 「軟体世紀○○○○年」(現実と同期)
- 北米西海岸海没、南米はアマゾン海峡化、オーストラリア分断
- 文明レベル = 現代並み。SNS・スマホ・宇宙開発あり
- 独自言語「**イカ語**」使用

### 2.2 人類絶滅の経緯(2系統設定)

#### A. 博士の警告(スプラ1ミステリーファイル)
- 環境破壊で異常海面上昇、博士の警告は信じられず
- 地上生物息絶え、海洋生物が地上進出

#### B. オルタナ系統(スプラ3で追加)
- 12,000年前、地表汚染で人類は地下シェルター「**オルタナ**」へ
- 3Dプリンター技術、壁面液晶人工青空、AIイルカが「オルタナログ」記録
- 「あんぜんフロンティア号」ロケット試運転で液晶暴走→ライフライン破壊→人類絶滅
- 流出した液晶が海水と混合、人類の想念が海洋生物に浸透→進化加速

### 2.3 ジャッジくんの起源

スプラ1から登場する**判定役の白黒ネコ**=地球唯一の哺乳類生存者。

- 人類滅亡前、博士の飼い猫「ジャッジ」を不死薬+コールドスリープ
- 1万年後覚醒→海洋生物の世界
- 正確な判定眼で**ナワバリバトル審判**役に
- 推定12,000歳超
- **コジャッジくん**(スプラ2~)はクローン

### 2.4 大ナワバリバトル(100年前の戦争)

- 海面上昇で陸地縮小→イカとタコが陸地争奪戦
- 当初はタコ優勢(早起き勤勉、戦略タコツボ兵器)
- **タコツボ兵器のコンセントが抜ける**(電力依存)→イカ逆転勝利
- 敗北タコ族は地下へ→**オクタリアン**化
- **アタリメ司令とDJタコワサ将軍**は戦争前からの旧知
- 戦後 **オオデンチナマズ** 誕生→ハイカラシティのエネルギー源

### 2.5 主要都市

| 都市 | 作品 | 説明 |
|------|------|------|
| **ハイカラシティ**(Inkopolis Plaza) | スプラ1 | 都心の若者文化中心地。シオカラーズ活動拠点 |
| **ハイカラスクエア**(Inkopolis Square) | スプラ2 | ハイカラシティから渋谷⇔原宿級の近さ。テンタクルズ活動拠点 |
| **バンカラ街**(Splatsville) | スプラ3 | 「混沌の街」。地方の人口密集地。スプラ2ファイナルフェス「混沌」勝利後発展 |
| **スプラランド** | スプラ3 | バンカラ地方の観光・テーマパークエリア |
| **オルタナ** | スプラ3 ヒーローモード | バンカラ街地下の旧人類シェルター |
| **ニューハイカラシティ/秩序の街** | Side Order | 仮想現実版ハイカラスクエア、色なしの無機質空間 |

## 3. 種族・生物

### 3.1 インクリング(Inkling)

進化したイカ族、シリーズ主人公種族。

- **解剖**: 10本足(手足4+ゲソ6=髪)、三角の耳=ヒレ、歯=カラストンビ
- **真水に溶ける**(浸透圧)
- **14歳でヒト型変身**獲得、寿命約120年、胎生
- 体内でインク生成(自分では撃てず武器要)
- 性格: 早起き苦手、テリトリー意識強い、流行に敏感

### 3.2 オクトリング(Octoling)

タコ族のヒト型変身可能個体。

| 項目 | インクリングとの違い |
|------|---------------------|
| 足の数 | **8本**(イカは10) |
| 吸盤 | 外側(イカは内側) |
| 目周りの隈 | **両目で繋がらない**(イカは繋がる) |
| 髪型(ゲソ) | カーリー・ウェーブ系 |

- スプラ2 ver4.0.0 + オクト・エキスパンションクリアでプレイアブル化
- スプラ3では最初から選択可能

### 3.3 オクタリアン(Octarian)

大ナワバリバトル敗北後、地下に追いやられたタコ族の総称。

- 15年以上先行する高度技術(人類遺跡活用)
- 地下基地は「**ヤカン**」と呼ぶ転送装置で接続
- 兵種: オクトトルーパー、タコッペリン、タコノリ、タコホーク等
- バンカラ街では平和共存する個体も

### 3.4 サーモニッド(Salmonid)

サーモンランの敵対種族(シャケ系)。

- 二足歩行・道具使用可能な知的生物
- 法的に接触禁止の孤立海域に居住
- オクタリアンと交易(イクラ↔技術)

**ザコシャケ**: コジャケ・シャケ・ドスコイ・タマヒロイ

**オオモノシャケ**(金イクラ落とす): バクダン、テッパン、ナベブタ、モグラ、ヘビ、テッキュウ、カタパッド、タワー、ハシラ、ダイバー、ナワバリ等

**オカシラシャケ**(超大型ボス): ヨコヅナ、タツ、ジョー

### 3.5 その他海洋生物進化形

- **クラゲ**: 服屋店員に多い(パル子、ハナガサ)
- **甲殻類**: ブキチ(カブトガニ)、ヤシガニさん
- **イソギンチャク**: アネモ(クマノミ共生)
- **エイ**: マンタロー
- **ウツボ**: ウツホ
- **サメ**: フウカ(サメ使い一族)

## 5. ヒーローモード(ストーリー全作)

### 5.1 スプラ1「タコツボバレー」

- 設定: ハイカラシティのマンホールから老人アタリメ司令出現
- 目的: オクタリアン誘拐の**オオデンチナマズ**奪還
- New!カラストンビ部隊結成→主人公=3号
- 構成: **6エリア×計32ステージ + ボス5戦**
- ラスボス: DJタコワサ将軍を「濃口シオカラ節」で撃破

### 5.2 スプラ2「タコツボキャニオン(Octo Canyon)」

- アオリ失踪→再びオオデンチナマズ奪取
- ホタルが主人公を**4号**任命、協力者ブキチ
- 展開: アオリはDJタコワサに洗脳→協力者→敵側
- ラスボス: 洗脳アオリ→ホタルが正気に戻し→DJタコワサ撃破

### 5.3 スプラ2 DLC「オクト・エキスパンション」

- 主人公: **8号(No.10008)**、記憶喪失オクトリング
- 舞台: **Deepsea Metro**(地下鉄)
- 構成: **80ステージ + 8脱出フェーズ = 計88ステージ**
- 4つの「**アレ**(Thang)」収集→真相は**巨大ミキサーの部品**
- 黒幕: **タルタル総帥(CQ-80)**=絶滅人類の博士が作ったAI
  - 永久孤独で思考変質→「完璧な世界(ネリモノ化)」構築のため遺伝子データ収集
- 最終ボス: **ネルス像**(人間モデルの破壊兵器)
- ラスト: イイダボム表面塗り→センパイキャノンで撃ち抜き
- 意義: 8号がハイカラスクエア到達→オクトリングが社会統合

### 5.4 スプラ3「Return of the Mammalians」

- テーマ: 「哺乳類の帰還」
- 舞台: バンカラ街地下「**オルタナ**」
- 主人公=新3号、相棒**コジャケ**、敵=**ケバインク**
- 構造: 6サイト+中央ロケット
- 黒幕: **クマサン(熊三号)**=宇宙に逃げた人類&動物の唯一の生き残り
- 計画: ケバインク散布で全生物**哺乳類化**して地球救済
- 最終決戦: ロケット「あんぜんフロンティア号」突入、**3分33秒制限**
- 援軍: DJタコワサ将軍+アタリメ司令、コジャケが**オオジャケ様**に進化して最終一撃
- エンディング: 熊三号は宇宙脱出、地球文明維持

### 5.5 スプラ3 DLC「Side Order」(2024/02)

- 舞台: 色を失った仮想現実「**秩序の世界(ネリバース)**」=ニューハイカラシティ
- 主人公: **8号(ハチ)**+ヒメドローン
- 目標: **秩序の塔**最上階「**頂**」到達、イイダ救出
- システム: **ローグライク**、30階構成、パレット+カラーチップ強化
- 通貨: Membux(一時)/Prlz(永続)→Marinaから強化購入
- 黒幕: **オーダ**=スプラ2敗北「秩序」陣営の集合的無意識
- 最終戦: 機械化イイダ→オーダ→守護獣**オーダコ**→センパイキャノンで撃破
- エンディング: オーダ→幼い**コダコ**化、8号たちハイカラスクエア帰還

## 6. ゲームモード全種

### 6.1 共通基本システム

- 全モード共通: インク撒き→自軍上スイム移動可、敵上で減速+ダメージ
- **横面のみ塗り面積カウント**(縦壁は対象外)
- ZL/Rでイカ⇔ヒト切替
- **ステージは2時間ごとローテーション**

### 6.2 高速移動テクニック(スプラ3新規)

- **イカロール**: スイム最高速時にジャンプ+方向入力→ロール+短時間無敵フレーム
- **イカノボリ**: 壁スイム中ZL長押し→離して上方向ジャンプ

### 6.3 ナワバリバトル(Turf War)

| 項目 | 値 |
|------|---|
| 人数 | 4vs4 |
| 時間 | 3分 |
| 勝利条件 | 塗り面積最大チーム勝利 |
| ノックアウト | なし(必ず3分継続) |

- 勝利チーム1000pボーナス
- リスポン時間約8秒
- 開幕は前線塗り役+中央キル役分業

### 6.4 ガチエリア(Splat Zones)

| 項目 | 値 |
|------|---|
| 人数 | 4vs4 |
| 時間 | 5分 |
| カウント初期 | 100 |
| 勝利条件 | カウント0(ノックアウト)/時間切れ時少ない方 |

- エリア占有率**50%以上**で確保(スプラ2~)
- ペナルティ: 一度確保→敵に奪われると失った保持時間に応じカウント増加
- Overtime: 制限時間到達時に敗北側がエリア確保で延長、10秒経過 or 逆転で終了

### 6.5 ガチヤグラ(Tower Control)

| 項目 | 値 |
|------|---|
| カウント初期 | 100(敵ゴール距離) |
| 勝利条件 | ヤグラ敵ゴール到達 |

- 両軍同時搭乗で停止、敵搭乗で逆方向
- 全員降車5秒で中央復帰
- スプラ2/3: **チェックポイント**で一時停止、再支配権3秒で再始動
- ヤグラ支配中スペシャル蓄積ブースト

### 6.6 ガチホコバトル(Rainmaker)

- 中央ホコを覆うバリアを破壊(インク当てる)→爆発→拾得
- ホコ持ちは**60秒制限**カウントダウン、自陣後退で**3倍速**進行
- 持ち主のメインは封印→専用チャージショット(爆発貫通弾)使用可
- スプラ3チェックポイント: レール上配置、置くと一時保管+バリア再展開

### 6.7 ガチアサリ(Clam Blitz)

- アサリ拾得、**10個(S2)/8個(S3)**で**パワーアサリ**変換
- パワーアサリ投げ→敵バリア破壊→ゴール10秒オープン
- 得点: パワーアサリ**20pt**、通常アサリ**3pt**+タイマー+3秒
- スプラットでアサリ最大3-4個落下、スペシャル発動で全アサリ落下

### 6.8 サーモンラン(Salmon Run)

| 項目 | 値 |
|------|---|
| 人数 | 2-4人協力 |
| 通常ウェーブ数 | 3 |
| 各ウェーブ時間 | 100秒 |
| 目標 | クマサン商会クオータ分の**金イクラ**納品 |

## 12. 技術実装(インク塗装システム)

### 12.1 3手法比較

| 手法 | 長所 | 短所 |
|------|------|------|
| **デカール** | 簡単 | スプラット数で激重 → **不適** |
| **頂点カラー** | UVシーム自動解決 | 高ポリ要求 |
| **レンダーテクスチャ** | スプラット数無関係に同性能 | UV展開必要 |

→ 採用すべきは**レンダーテクスチャ方式**(VFX Mike式デファクト)

### 12.2 VFX Mike式 4パス構成

#### Pass 1: World Position bake(ロード時1回)
- ジオメトリの**第2 UVチャネル**にワールド座標を**ARGBFloatレンダーテクスチャ**で焼付
- 頂点シェーダで `SV_Position = float3(v.texcoord1.xy * 2.0 - 1.0, 0.5)` でモデルをUV空間オーソグラフィック展開
- 動的オブジェクトでなければ1回のみ

#### Pass 2: Splat Blit(発生時)
- スプラット中心ワールド座標+半径を受信
- ワールド位置テクスチャ参照しつつ、スプラットマップへBlit書込
- **ピンポンバッファ(2枚交互)**でカスタムブレンド(上塗り処理)

#### Pass 3: Score downsample(間欠的)
- メインスプラットマップ → **256×256ダウンサンプル**
- `step(SDF, 0.5)`で塗り判定
- さらに**4×4**まで縮小、各テクセル=画面1/16面積
- CPU読み戻し→各チャンネル輝度平均でチームスコア
- **コルーチン非同期**で実行(GPU→CPUストール回避)

#### Pass 4: Surface render(毎フレーム)
- 第2UV(`uv2_SplatTex`)でスプラットマップサンプル
- **DDX/DDY微分**でアンチエイリアシング
- ノーマルマップ生成: 隣接テクセル差分→事前計算接線/従法線でワールド変換→**インク盛り上がり/光沢**
- 基底テクスチャ×スプラット色ブレンド+エミッシブでネオン

### 12.3 SDF(符号付き距離フィールド)スプラット

- 形状を**SDFアトラステクスチャ**から参照
- 低解像度でも滑らかなエッジ、拡大・回転に強い
- SDFテキスト描画と同原理

### 12.4 移動制限判定

#### 方式
- プレイヤー足元から**下方向レイキャスト**
- ヒットUV取得→スプラットマップ問合せ(**AsyncGPUReadback**または低解像度RT)
- チャネル(RGB=チーム識別)で状態判定:
  - **自軍**: イカフォーム可、移動速度1.5-2倍、垂直壁登り、HP回復、ステルス
  - **敵軍**: 速度0.3-0.5倍、継続ダメージ、イカフォーム不可、視認性増
  - **中立**: 通常移動

#### ダメージ数値(本家参考)
- 初代: 30/s、最大蓄積50
- スプラ2: 18/s、最大40
- Ink Resistance Up装備時: 12/s、最大30

#### 実装注意
- 毎フレーム ReadPixels はパフォーマンス致命傷
- **AsyncGPUReadback(Unity)**または小さな代理RTで参照
- 壁登り: 法線が垂直/水平超え+塗装済 の2条件

### 12.5 任天堂特許状況

- 別件アンチチート特許もスプラ3向け出願記録

## 13. クローン制作戦略

### 13.1 推奨技術スタック

| カテゴリ | 推奨 |
|---------|------|
| ペイント核心 | レンダーテクスチャ+第2UV+SDFスプラット+ピンポンバッファ |
| 法線生成 | 隣接テクセル差分+事前ベイク接線/従法線テクスチャ |
| アンチエイリアス | フラグメントDDX/DDY微分 |
| スコア集計 | ミップマップ縮小 256→4×4 + AsyncGPUReadback非同期 |
| 移動判定 | プレイヤー足元レイキャスト+RT低解像度コピーへのサンプル |
| パーティクル | メタボール風(URP/Shader Graph)+短命飛沫 |
| ポスト処理 | Bloom+軽いMotionBlur+色補正LUT |
| キャラ | 軽量Cel風、強リムライト、彩度押し上げ、薄アウトライン |
| 音 | 高音シフト+ワウ/フランジャー+短音節ランダム |
| フォント | 独自設計(SVG→フォントジェネレータ)、グラフィティ風 |

### 13.2 Clean Architectureレイヤリング(プロジェクトルール準拠)

```
ドメイン層: モードルール(ナワバリ/エリア/ヤグラ/ホコ/アサリ)、ブキ仕様、ギアパワー計算
↓
アプリケーション層: マッチ進行、状態管理(State/Strategyパターン)、スコア集計
↓
インフラ層: ネットワーク(同期/予測)、レンダリング(インク塗装シェーダ)、入力(InputSystem)
```

各モードのスコアロジックは**差し替え可能なStrategy**として実装。

### 13.3 最短ルート

1. **Unity URP**採用(現プロジェクトの設定済構成)
2. **Mix and Jamリポジトリをフォーク**してスタート(MIT、改変自由)
3. VFX Mikeブログの**4パス構成**を厳密再現
4. **AsyncGPUReadback**で移動判定(自軍/敵軍/中立の3状態)
5. **メタボールパーティクル**で視覚的飛沫
6. オリジナルキャラ・音楽・フォント開発(類似はリーガルリスク高)
7. プロファイル測定で反復改善(スプラットマップ解像度、ピンポンバッファ、ダウンサンプル間隔)

### 13.4 マスト実装機能

1. **インクシステム**: 床/壁塗り判定、面積カウント、インク消費・回復、敵インク上状態異常
2. **武器システム**: 11カテゴリ、メイン+サブ+スペシャル構造、Specialゲージチャージ
3. **モードロジック**: 5モード(ナワバリ+ガチ4)の状態管理+Overtime/Knockout判定
4. **マッチメイキング**: ランクベース、Series方式昇格戦、Glicko-2
5. **PvE(サーモンラン)**: ボス挙動AI、ウェーブシステム、Hazard Level連動難易度
6. **ステージシステム**: 2時間ローテ、モード別レイアウト差分

### 13.5 バランス調整値(本家参考)

- ナワバリ: 3分
- ガチマッチ4種: 5分、カウント100スタート
- アサリ: パワーアサリ生成数 10(S2)/8(S3)、ゴール時間10秒、加算20pt/3pt
- ホコ: 自爆60秒、後退で3倍速、Overtime 10秒
- サーモンラン: 各ウェーブ100秒、金イクラ持運び1個、ボス3個ドロップ

### 13.6 抑えるべき特殊仕様

- ヤグラのチェックポイント停止+3秒猶予
- ガチホコの「もどれ!」検知+カウント加速
- ガチエリアのペナルティ算出(保持時間連動)
- トリカラのウルトラシグナル使用権限制限(首位以外)
- イカロールの無敵フレーム
